# JobSearchBot

- Custom job search database designed to automatically find open positions and save them to a local database.
- Will allow for saving jobs to apply to later or after applying for reference.
- Can delete jobs to avoid bloat.
- Searches most common job sites.
- Allows for simple tracking of the job and status.
- Attempts to match jobs from multiple sources to avoid duplication.
- Creates scheduled digests of newly opened jobs.
- Allows for customization of job location and in-office requirements (i.e. remote, hybrid, on-site)
